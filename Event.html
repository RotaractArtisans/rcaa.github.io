Our Projects

Each and Every Project we work on is an initiative to make a difference in people's life .Help those in need . Doing all possible things to serve the community in a cost effective yet beneficial in the long run way . 

Project Adhikar

Project Adhikar     

About the project

This project was started with Govt of India and we organised camps for Voter Identification and Aadhar card registrations of people.This project is related to the registrations of citizens for different identifications like VoterId and PAN cards.

Project Inayat

Project Inayat      

About the project

.An initiative to bring smiles, and joy on the faces of people. Under this Project, we visit various orphanages, slums and spend quality time with the kids, old aged parents of these organisations.In this project we meet children in the orphanage and people in the old age home and try to spend time with them and make them feel special

Project Postcard

Project Postcard  

About the Project

An initiative to bring a sense of old times through a hand written card which can express emotions beyond description.
We make handmade customised postcards according to the upcoming festival.

Project Karigari

Project Karigari

About the project

An initiative to teach the underprivileged children with major skills through workshops. Under this Project, we have collaborated with a few MCD schools in Delhi.The project is about teaching the underprivileged kids art and crafts skills and also soft skills.

Project Wings

Project Wings        

About the Project

An initiative to let people break the taboos in the society and Through this project., We focus on fighting various diseases prevailing in the society by making people aware about sex education, Menstruation, Child Trafficking, etc.In this project we teach and aware people about the topics that are considered taboos in the society or are major issues related to women.

Project Neeti

Project Neeti        

About the Neeti

An initiative to spread the legal awareness among the generations of the society. In our country very few people are aware of their rights, specially the backward section So through this project we help people to know more .This project is about teaching the kids the basic laws that are indispensable in our life yet are not given much importance.

Project Artisans(District Project)

Project Artisans
